# DDI 2.5 to fo-xml (to be trasformed to PDF)

To run you need Apache FOP:
https://xmlgraphics.apache.org/fop/download.html

To run:

`/path/to/fop -xml path/to/my-ddi-2.5.xml -xsl path/to/dditofo.xsl -pdf my-pdf.pdf`
